# AgentSearch Connector

AgentSearch-style systems provide a self-hosted search and read API for agents.

Useful capabilities:

- multi-engine search
- page content extraction
- deduplication
- caching
- content scrubbing
- MCP wrapper

Reference:

- [SearXNG](https://github.com/searxng/searxng)

This connector is optional. Newswiki can also use RSS, manual URLs, or browser automation.
