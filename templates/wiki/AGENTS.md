# Private Newswiki Instance Rules

## Startup Protocol

Before non-trivial work:

1. Query Capability MCP for relevant tools and skill chains.
2. Query Wiki MCP with `wiki_past_knowledge(task)`.
3. Query Newsfeed MCP if current information matters.

## Write-Back Rule

Write back only durable learning:

- reusable pattern
- decision
- pitfall
- gap

Do not write raw logs or uncertain one-off notes into control pages.

## Privacy

This private instance may contain personal information. Do not push it to a public repository.
